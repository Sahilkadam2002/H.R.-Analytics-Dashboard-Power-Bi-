# H.R.-Analytics-Dashboard-Power-Bi-
This repository contains an interactive HR Analytics Dashboard built in Power BI, designed to help organizations monitor key workforce metrics and make informed HR decisions. The dashboard provides a comprehensive view of employee data, enabling insights into attrition, hiring trends, performance, and workforce demographics. 
## Dataset 
<a href="https://github.com/Sahilkadam2002/H.R.-Analytics-Dashboard-Power-Bi-/blob/main/POWER%20BI%20WORKING.pbix">Dataset</a>
## KPI's 
* Employee Attrition Analysis (overall, by department, age group, job role)
* Workforce Demographics (gender, education, age, marital status)
* Department-Level Insights on headcount, attrition rate, and job satisfaction
* Performance & Tenure Metrics highlighting employee experience and engagement 
* Dynamic Slicers & Filters for interactive and deep exploration  
* Added a chart where employees designation and the job satisfaction which they have faced are showing. 
* Average salary chart is in the dashboard to show the salary range.
